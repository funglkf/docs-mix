# Home

## Quick Start on localhost

!!! Note

    === "Global Python"

        ```
        pip install -r requirements.txt
        mkdocs serve
        ```

    === "Virtual environment"

        ```
        source env/Scripts/activate
        pip install -r requirements.txt
        mkdocs serve
        ```

## Qick deploy to Github (Now by github action)
`mkdocs gh-deploy --force`
